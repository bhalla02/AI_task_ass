# Probability — JEE Level

## 1. Basic Definitions

Probability:
P(A) = (Number of favorable outcomes) / (Total outcomes)

0 ≤ P(A) ≤ 1

---

## 2. Addition Rule

P(A ∪ B) = P(A) + P(B) - P(A ∩ B)

If mutually exclusive:
P(A ∩ B) = 0

---

## 3. Multiplication Rule

P(A ∩ B) = P(A) P(B|A)

If independent:
P(A ∩ B) = P(A) P(B)

---

## 4. Conditional Probability

P(A|B) = P(A ∩ B) / P(B)

---

## 5. Bayes Theorem

P(A|B) = [P(B|A) P(A)] / P(B)

---

## 6. Permutations & Combinations

nPr = n! / (n-r)!
nCr = n! / (r!(n-r)!)

---

## 7. Binomial Probability

Probability of exactly r successes in n trials:

P = nCr * p^r * (1-p)^(n-r)

---

## 8. Common Mistakes

- Not checking independence.
- Using permutation instead of combination.
- Forgetting complement rule: P(A') = 1 - P(A)
