# Basic Calculus — JEE Level

## 1. Limits

Basic limits:

lim (x → 0) sin x / x = 1
lim (x → 0) (1 - cos x)/x^2 = 1/2
lim (x → 0) (e^x - 1)/x = 1

L'Hospital's Rule:
If limit gives 0/0 or ∞/∞,
differentiate numerator and denominator.

---

## 2. Derivative Rules

d/dx (x^n) = n x^(n-1)
d/dx (sin x) = cos x
d/dx (cos x) = -sin x
d/dx (e^x) = e^x
d/dx (ln x) = 1/x

---

## 3. Product Rule

d/dx (u v) = u dv/dx + v du/dx

---

## 4. Chain Rule

If y = f(g(x)):

dy/dx = f'(g(x)) * g'(x)

---

## 5. Optimization Strategy

1. Define variable.
2. Express function to optimize.
3. Differentiate.
4. Set derivative = 0.
5. Check second derivative or boundary.

---

## 6. Common Mistakes

- Forgetting chain rule.
- Not checking critical points.
- Ignoring domain constraints.
